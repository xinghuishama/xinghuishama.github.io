## 核心功能

| 模块 | 功能说明 |
|------|----------|
| **智能分析** | 输入号码或生肖，实时统计 1-49 号频次，支持 Worker 多线程加速 |
| **杀码系统** | 自定义斩杀号码，自动扣减命中次数，上限 6 次 |
| **多维筛选** | 生肖、头数、尾数、数段、波色、五行、半单双、合数 8 大维度 |
| **实时开奖** | 自动获取当期开奖号码，21:33:30 起每 5 秒刷新，开完七码自动停止 |
| **历史开奖** | 按年份分页查询，支持 2020 年至今全部期数 |
| **开奖直播** | 直播时段自动加载视频源，支持 HLS/FLV/API 三源自动切换 |
| **独苗特效** | 当仅剩 1 个未命中号码时，触发飞入守护动画 |
| **粒子背景** | 彩色水泡粒子自下而上升起，透明背景可见飞升效果 |
| **自动跨年** | 生肖映射基于算法自动生成，无需每年手动改代码 |

## 自动跨年生肖算法
基准年：2024 = 龙年 生肖序列：龙 蛇 马 羊 猴 鸡 狗 猪 鼠 牛 虎 兔
index = (year - 2024) % 12
| 年份 | 太岁 | 01 号归属 |
|------|------|-----------|
| 2024 | 龙 | 龙 |
| 2025 | 蛇 | 蛇 |
| 2026 | 马 | 马 |
| 2027 | 羊 | 羊 |

系统每年自动识别当前年份，生成正确的生肖-号码映射，**永不过期**。

## 技术栈

- **HTML5** + **Tailwind CSS**（CDN）
- **Vanilla JavaScript**（无框架依赖）
- **Web Worker**（独立分析引擎，防 UI 阻塞）
- **Canvas 2D**（粒子特效）
- **Hls.js / flv.js**（直播流播放）
- **GitHub Pages**（静态部署）

## 本地使用

1. 下载全部 5 个文件到同一目录
2. 用浏览器打开 `index.html`
3. 无需服务器，无需构建，即开即用

> 注意：由于浏览器安全策略，部分功能（如 Worker、本地存储）在 `file://` 协议下可能受限，建议通过本地服务器或 GitHub Pages 访问。

---

## 部署到 GitHub Pages

1. 新建 GitHub 仓库（如 `mashen`）
2. 上传 `index.html`、`style.css`、`data.js`、`app.js`、`worker.js`
3. 进入仓库 **Settings → Pages**
4. Source 选择 **Deploy from a branch**，分支选 `main`，目录选 `/ (root)`
5. 保存后等待 1-2 分钟，访问 `https://你的用户名.github.io/mashen/`
