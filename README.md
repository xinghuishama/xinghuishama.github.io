## 核心算法

### 五行自动跨年

基于 **60甲子纳音五行** + **30位周期序列**：

```javascript
const WUXING_BASE_SEQ = [
  '金','金','土','土','木','木','火','火','金','金',
  '水','水','木','木','火','火','土','土','水','水',
  '木','木','金','金','土','土','水','水','火','火'
];

// 每年序列整体右移1位，01号五行 = 当年纳音五行
function getNumberWuxing(num, year) {
  const idx = (num - 1) % 30;          // 周期30
  const offset = year - 2023;           // 相对基准年偏移
  return WUXING_BASE_SEQ[(idx - offset + 30) % 30];
}
```

**验证：**
| 年份 | 干支 | 纳音 | 01号五行 |
|:---:|:---:|:---:|:---:|
| 2023 | 癸卯 | 金箔金 | **金** |
| 2024 | 甲辰 | 佛灯火 | **火** |
| 2025 | 乙巳 | 佛灯火 | **火** |
| 2026 | 丙午 | 天河水 | **水** |
| 2027 | 丁未 | 天河水 | **水** |
| 2028 | 戊申 | 大驿土 | **土** |
